# WarmPaws – Pet Care in Winter

**Live Demo:** [[https://warmpasw-pet-cere.web.app](https://warmpaws-pet-care.web.app/)]

---

## Project Overview
WarmPaws is a cozy winter companion platform designed for pet owners to ensure their furry friends stay warm, safe, and healthy during the cold season. Users can explore local pet care services, winter pet clothing, grooming options, and expert tips — all in one friendly interface.

---

## Key Features

- Responsive design for mobile, tablet, and desktop.
- Authentication with Email/Password and Google social login.
- View winter care services with details like price, rating, and category.
- Book services via a simple form with success notification.
- User profile page with update functionality.
- Extra sections: Winter care tips for pets, Meet our expert vets, Local winter pet events.
- Smooth SPA routing with React Router.
- Subtle animations using AOS, Animate.css, and Swiper.js.
- Toast notifications for success and error messages.

---

## NPM Packages Used

- react: Core library for building UI.
- react-dom: Rendering React components.
- react-router: SPA routing for navigation between pages.
- firebase: Authentication, hosting, and backend services.
- tailwindcss: Styling framework for responsive UI.
- @tailwindcss/vite: Tailwind plugin for Vite integration.
- react-hot-toast: Toast notifications for success and error messages.
- react-icons: Ready-made icons for UI components.
- swiper: Hero slider component for winter pets showcase.
- react-spinners: Loading spinners for page transitions.
- sweetalert2: Modal popups for alerts and confirmations.


---

## Purpose

The purpose of this project is to provide a friendly and safe platform for pet owners during the winter season. WarmPaws helps users:

- Keep their pets warm and healthy with curated winter care services.
- Explore grooming options, winter clothing, and essential accessories for pets.
- Access tips and expert advice for pet care in cold weather.
- Book local services easily through a simple interface.
- Maintain a personal profile and track booked services.

Overall, the goal is to ensure a comfortable and worry-free winter experience for pets and their owners.
